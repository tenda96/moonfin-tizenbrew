# Moonfin TizenBrew

Experimental TizenBrew application module for Moonfin Smart TV.

This repository repackages the official Moonfin Tizen `.wgt` release as a TizenBrew-compatible GitHub module.

## Install in TizenBrew

Try adding this module in TizenBrew:

```text
gh:tenda96/moonfin-tizenbrew
```
